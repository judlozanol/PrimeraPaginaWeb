CSS

  • CSS son las siglas en inglés para "hojas de estilo en cascada" (Cascading Style Sheets).
  • Es el encargado de toda la capa de personalización sobre el que el sitio web define su aspecto final.
  • Para más información sobre CSS consultar aquí.

JS

JavaScript (JS) es un lenguaje de programación que permite implementar funciones complejas en páginas web. Se ejecuta en el lado del cliente de la web, y se puede utilizar para estilizar/programar cómo se comportan las páginas web cuando ocurre un evento. Más información sobre este lenguaje puede ser consultada aquí.
